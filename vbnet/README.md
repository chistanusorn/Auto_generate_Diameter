# Auto Generate Diameter Sheet

ระบบสร้างรายงาน Diameter ด้วย VB.NET, .NET 8 และ Windows Forms

## 1. Overviews

โปรเจคนี้ใช้สร้างรายงาน Diameter จากข้อมูลที่นำเข้าจาก MySQL หรือไฟล์ XLSX แล้วจัดวางข้อมูลลงแบบฟอร์ม T-1 ถึง T-22 ตาม logic ของ report

ความสามารถหลัก:

- โหลดข้อมูลจาก MySQL ด้วย Production Place Code และ Coat Lot Number
- Import ข้อมูลจากไฟล์ XLSX
- จัดข้อมูลลงช่อง T-1 ถึง T-22 และรองรับหลายหน้า
- ให้ user กรอก/แก้ Header เอง เช่น ITEM, PLATE No., DATETIME, CAPA, DOME Type, Coat Lot No.
- เลือกตำแหน่ง T-* เริ่มต้นของข้อมูลได้
- Zoom ด้วย mouse wheel และ pan ด้วยการลากเมาส์ซ้าย
- Export ข้อมูลออกเป็น CSV
- สร้าง Installer สำหรับแจกภายในบริษัทด้วย Inno Setup

Technology:

- VB.NET และ .NET 8
- Windows Forms
- MySqlConnector
- ClosedXML
- MSTest
- Inno Setup 6

## 2. โครงสร้างโปรเจค

```text
vbnet/
├── AutoGenerateDiameter.sln
├── AutoGenerateDiameter.Core/
├── AutoGenerateDiameter.WinForms/
├── AutoGenerateDiameter.Tests/
├── installer/
├── publish/
├── build-installer.ps1
└── README.md
```

### `AutoGenerateDiameter.sln`

Solution หลักสำหรับเปิดโปรเจคทั้งหมดใน Visual Studio ประกอบด้วย Core, WinForms และ Tests

### `AutoGenerateDiameter.Core`

เก็บ models และ business logic ที่ไม่ผูกกับหน้าจอ

| ไฟล์ | หน้าที่ |
|---|---|
| `Models.vb` | โครงสร้างข้อมูล เช่น `Measurement`, `SheetData`, `SourceRecord`, MySQL settings |
| `MySqlReportLoader.vb` | เชื่อมต่อ MySQL, รัน SQL, โหลดข้อมูลสำหรับสร้าง report |
| `XlsxReportImporter.vb` | อ่านและ map ข้อมูลจากไฟล์ XLSX |
| `ReportBuilder.vb` | จัดกลุ่ม tray และสร้าง sheet ตาม logic report |
| `EmptyReportFactory.vb` | สร้าง report ว่างตอนเปิดโปรแกรม |
| `SampleReportFactory.vb` | สร้างข้อมูลตัวอย่างสำหรับพัฒนา/ทดสอบ |

### `AutoGenerateDiameter.WinForms`

เก็บ Desktop UI และการเชื่อม action จาก user ไปยัง Core

| ไฟล์ | หน้าที่ |
|---|---|
| `Program.vb` | จุดเริ่มต้นโปรแกรม |
| `Form1.vb` | หน้าหลัก, toolbar, load/import/export, page state |
| `Form1.Designer.vb` | ส่วนประกอบพื้นฐานของหน้าหลัก |
| `Dialogs.vb` | Dialog สำหรับ MySQL, Header, Operator/start T |
| `ReportPreviewControl.vb` | วาด report, table, header/footer, zoom/pan |

ปุ่มหลัก:

| ปุ่ม | หน้าที่ |
|---|---|
| `Load Database` | โหลดข้อมูลจาก MySQL |
| `Import XLSX` | โหลดข้อมูลจากไฟล์ XLSX |
| `Edit Header` | แก้ Header และ Operator |
| `Adjust Start T` | เปลี่ยนตำแหน่ง T-* เริ่มต้นของข้อมูล |
| `Export CSV` | Export ข้อมูล report เป็น CSV |
| `< Previous` / `Next >` | เปลี่ยนหน้า report |

### `AutoGenerateDiameter.Tests`

เก็บ unit tests ของ logic หลัก

| ไฟล์ | สิ่งที่ทดสอบ |
|---|---|
| `ReportBuilderTests.vb` | Logic จัดกลุ่ม tray และสร้าง report |
| `XlsxReportImporterTests.vb` | การ import และ map XLSX |
| `MySqlReportLoaderTests.vb` | การโหลดข้อมูล MySQL |
| `EmptyReportFactoryTests.vb` | การสร้าง report ว่าง |

### `installer` และ `build-installer.ps1`

- `installer/AutoGenerateDiameter.iss` คือ Inno Setup script
- `installer/output/AutoGenerateDiameterSetup-x64.exe` คือ installer ที่สร้างเสร็จ
- `build-installer.ps1` ใช้ publish โปรแกรมและ compile installer อัตโนมัติ

### โฟลเดอร์ที่สร้างอัตโนมัติ

- `bin/` และ `obj/`: ไฟล์จากการ build
- `publish/`: ไฟล์โปรแกรมสำหรับแจก
- `verify_output/`: output ชั่วคราวสำหรับตรวจ compile

## 3. Flow การทำงาน

### Flow หลัก

```text
เปิดโปรแกรม
    ↓
กรอก Operator และเลือก Start T
    ↓
เลือกแหล่งข้อมูล
    ├─ Load Database
    └─ Import XLSX
    ↓
Core แปลงข้อมูลเป็น SourceRecord / Measurement
    ↓
ReportBuilder จัดกลุ่มข้อมูลตาม tray logic
    ↓
สร้าง SheetData หน้า 1..N
    ↓
ReportPreviewControl แสดงรายงาน
    ↓
user zoom, pan, เปลี่ยนหน้า, edit header, adjust start T หรือ export CSV
```

### Flow: Load Database

```text
กด Load Database
    ↓
กรอก Host, Port, Database, Username, Password
    ↓
กรอก PPC และ CLN
    ↓
MySqlReportLoader รัน SQL
    ↓
ReportBuilder จัดข้อมูลลง T-1..T-22
    ↓
แสดงรายงาน
```

### Flow: Import XLSX

```text
กด Import XLSX
    ↓
เลือกไฟล์ .xlsx
    ↓
XlsxReportImporter อ่านชื่อ column และข้อมูล
    ↓
map เป็น SourceRecord
    ↓
ReportBuilder สร้าง report ใหม่แทนข้อมูลเดิม
    ↓
แสดงรายงาน
```

### Flow: Edit Header / Adjust Start T

```text
กด Edit Header หรือ Adjust Start T
    ↓
user กรอกค่าที่ต้องการ
    ↓
โปรแกรม update sheet ทุกหน้า
    ↓
ถ้าเปลี่ยน Start T จะ reposition ข้อมูลโดยคง label T-1..T-22 ไว้เหมือนเดิม
```

### Flow: Export CSV

```text
กด Export CSV
    ↓
เลือกตำแหน่งไฟล์
    ↓
โปรแกรมเขียนข้อมูลทุกหน้าเป็น CSV
    ↓
แสดง popup เมื่อ export สำเร็จหรือ error
```

### Flow: สร้าง Installer

ต้องติดตั้ง Inno Setup 6 ที่ `D:\Inno Setup 6`

```powershell
cd D:\hoya\Auto_generate_diameter_sheet\vbnet
.\build-installer.ps1 -Version "1.1.0"
```

ลำดับการทำงาน:

```text
dotnet publish แบบ Release และ self-contained
    ↓
สร้างไฟล์ใน publish/win-x64
    ↓
Inno Setup compile installer
    ↓
ได้ไฟล์ installer ใน installer/output
```
