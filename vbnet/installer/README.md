# Inno Setup

สร้าง Installer:

```powershell
cd D:\hoya\Auto_generate_diameter_sheet\vbnet
.\build-installer.ps1
```

กำหนด Version:

```powershell
.\build-installer.ps1 -Version "1.0.1"
```

ไฟล์ผลลัพธ์:

```text
installer\output\AutoGenerateDiameterSetup-x64.exe
```

Installer จะ:

- ติดตั้งแบบ self-contained ลง `Program Files`
- รวม `coat_lot_query.sql` และ dependency ทั้งหมด
- สร้าง Start Menu shortcut
- เลือกสร้าง Desktop shortcut ได้
- รองรับ Uninstall ผ่าน Windows Settings
